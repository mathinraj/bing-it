## Features

| | Feature | Description |
|---|---|---|
| ⌨️ | **Human-like Typing** | Characters typed one-by-one with variable delays (45–580 ms) and occasional "thinking" pauses |
| 📜 | **Natural Scrolling** | Scrolls search results in random increments with reading pauses |
| ⏱️ | **Floating Timer** | Live countdown overlay on the Bing tab showing progress and time to next search |
| ☕ | **Coffee Breaks** | Configurable long pauses between searches for natural pacing |
| 📈 | **Google Trends** | Real-world trending queries so your search history looks 100% natural |
| 🗂️ | **500+ Fallback Queries** | Diverse built-in pool across 20+ categories when Trends is unavailable |
| ⚙️ | **Fully Configurable** | Control delays, break frequency, break duration, and reading overlap |
| 🔄 | **Background Execution** | Searches run in an inactive tab — keep using your browser normally |

### Manual Install (Developer Mode)

1. Clone or download this repository
2. Open your browser's extension page:
   - **Chrome** → `chrome://extensions/`
   - **Edge** → `edge://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked** → select the project folder
5. Pin the extension to your toolbar

---

## Usage

1. **Sign in** to your Microsoft account in the browser
2. Click the extension icon → set search count → **Bing it!**
3. A background Bing tab opens with a **floating timer overlay**
4. Open **Settings** to customize delays, coffee breaks, and scrolling behavior
5. Click **Stop** at any time to cancel
